# Blogr Landing Page

A responsive landing page with interactive components.

## Features

- **Accessibility** — keyboard support, ARIA attributes, semantic HTML
- **Dropdown navigation**
- **Mobile navigation menu**
- **Responsive design** — adapts to different device widths
